## Errores comunes

<div style="background:#FEE2E2;border:1px solid #FECACA;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:12px;font-weight:600;color:#991B1B;margin:0 0 0.5rem;">"El nombre del operador es requerido"</p>
<p style="font-size:12px;color:#991B1B;margin:0;">Dejaste el nombre vacío. Escribe el nombre completo.</p>
</div>

<div style="background:#FEE2E2;border:1px solid #FECACA;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:12px;font-weight:600;color:#991B1B;margin:0 0 0.5rem;">"El formato del email no es válido"</p>
<p style="font-size:12px;color:#991B1B;margin:0;">El email tiene formato incorrecto. Verifica que tenga formato <span style="font-family:'JetBrains Mono',monospace;font-size:11px;">nombre@dominio.com</span></p>
</div>

<div style="background:#FEE2E2;border:1px solid #FECACA;border-radius:0.75rem;padding:1rem;">
