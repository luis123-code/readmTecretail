<span style="font-family:'Cormorant Garamond',Georgia,serif;font-style:italic;font-size:18px;color:#6B6B6B;line-height:1.6;display:block;margin-bottom:1.25rem;">Herramienta de búsqueda multicampo para localizar notas de crédito específicas por cliente, documento de identidad o comprobante relacionado, con filtrado en tiempo real y compatibilidad con filtros de fecha.</span>

<alert type="info">
  El buscador para notas de crédito incluye el campo <strong>"Comprobante"</strong> para buscar por el documento original (factura/boleta) que corrige la nota. Es el campo más preciso para verificar si existe una nota de crédito para un comprobante específico.
</alert>
