## Cómo Usar el Filtro

<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">Fecha inicial del rango. El sistema mostrará facturas desde esta fecha en adelante (inclusive).</p>
</div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Cómo seleccionar</p>

<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#E5E5E5;color:#6B6B6B;flex-shrink:0;">01</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">Haz clic en el campo <span style="font-weight:600;color:#121212;">"Desde"</span></span></div>
<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#E5E5E5;color:#6B6B6B;flex-shrink:0;">02</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">Se abre <strong>calendario</strong> desplegable</span></div>
<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#E5E5E5;color:#6B6B6B;flex-shrink:0;">03</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">Navega al <span style="font-weight:600;color:#121212;">mes y año</span> necesarios</span></div>

<p style="font-size:12px;color:#6B6B6B;margin:0.5rem 0 0;line-height:1.5;"><strong>Resultado:</strong> El sistema mostrará facturas desde el 1 de enero en adelante</p>

<p style="font-size:12px;color:#92400E;margin:0;line-height:1.5;">Muestra facturas desde esa fecha <strong>hasta hoy</strong>. Útil para ver facturas recientes desde un punto específico.</p>
