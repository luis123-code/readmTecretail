## ¿Qué es el Filtro por Estado de Orden?

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

Muestra solo las cuentas por cobrar vinculadas a órdenes de venta con un estado determinado. Puedes combinar varios estados a la vez.

**Por defecto**, el sistema excluye automáticamente las órdenes <span style="display:inline-flex;align-items:center;padding:0.1rem 0.5rem;border-radius:9999px;background:#FEE2E2;border:1px solid #FECACA;font-size:11px;color:#DC2626;font-weight:500;">Anulada</span> para protegerte de gestionar cobros de ventas canceladas.

El filtro actúa en **tiempo real** — al marcar o desmarcar un estado la tabla se actualiza instantáneamente.

</div>
