## ¿Qué es la Lista de Clientes?

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

Vista dentro del módulo de Cobranza que agrupa todas las cuentas por cobrar **por cliente**. De un vistazo puedes ver cuánto debe cada cliente, cuántas cuentas tiene abiertas y en qué estado se encuentran.

Es tu herramienta principal para **priorizar cobranza**, **evaluar crédito** antes de nuevas ventas, y **analizar la concentración** de tu cartera.

</div>
<div style="background:#F9FAFB;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1.25rem;">

**Cómo acceder**

## ¿Cómo Usar la Lista?

<tabs>
  <tab title="Revisar">

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1rem 0;">
<div>

**Revisión general de cartera:**

1. Accede a la vista **"Por Cliente"**
2. Revisa la columna **Monto por Cobrar**
3. Identifica clientes con montos altos
4. Identifica clientes con cuentas vencidas
5. Prioriza según urgencia y monto

</div>
<div style="background:#F9FAFB;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0.5rem;font-weight:500;">Qué observar primero:</p>
<div style="display:flex;flex-direction:column;gap:0.4rem;font-size:12px;color:#121212;">
<span>1. Clientes con <span style="display:inline-flex;padding:0.05rem 0.4rem;border-radius:9999px;background:#FEE2E2;border:1px solid #FECACA;font-size:10px;color:#DC2626;">Vencido</span></span>
<span>2. Clientes con monto por cobrar alto</span>
<span>3. Clientes con muchas cuentas abiertas</span>
<span>4. Clientes sin actividad de pago</span>
</div>
</div>
</div>

  </tab>
  <tab title="Buscar">

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1rem 0;">
<div>

**Encontrar un cliente específico:**

1. Localiza el **campo de búsqueda**
2. Escribe el nombre o RUC del cliente
3. La lista se filtra en **tiempo real**
4. Haz clic en el cliente para ver el detalle de sus cuentas

</div>
<div style="background:#F9FAFB;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0.5rem;font-weight:500;">Puedes buscar por:</p>
<div style="display:flex;flex-direction:column;gap:0.4rem;font-size:12px;color:#121212;">
<span>→ Nombre completo o parcial</span>
<span>→ Razón social</span>
<span>→ RUC (20 dígitos)</span>
<span>→ DNI (8 dígitos)</span>
</div>
</div>
</div>

  </tab>
  <tab title="Detalle">

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1rem 0;">
<div>

**Ver cuentas de un cliente:**

1. Haz clic en el nombre del cliente
2. Se expanden o cargan las cuentas de ese cliente
3. Puedes ver el detalle de cada cuenta individual
4. Útil para gestión caso a caso

</div>
<div style="background:#F0FDF4;border:1px solid #A7F3D0;border-radius:0.75rem;padding:1rem;">
<p style="font-size:12px;color:#065F46;margin:0 0 0.5rem;font-weight:500;">En el detalle verás:</p>
<div style="display:flex;flex-direction:column;gap:0.4rem;font-size:12px;color:#065F46;">
<span>✓ Cada cuenta por cobrar del cliente</span>
<span>✓ Estado de cada cuenta individual</span>
<span>✓ Fecha de emisión y vencimiento</span>
<span>✓ Monto y pagos por cuenta</span>
</div>
</div>
</div>

  </tab>
  <tab title="Filtrar">

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1rem 0;">
<div>

**Combinar con los filtros globales:**

Los filtros de la barra superior afectan la lista de clientes. Puedes activar filtros de estado CxC, fecha de vencimiento o estado de orden para acotar qué clientes y qué cuentas se muestran.

</div>
<div style="background:#F9FAFB;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0.5rem;font-weight:500;">Combinaciones útiles:</p>
<div style="display:flex;flex-direction:column;gap:0.4rem;font-size:12px;color:#121212;">
<span>→ Lista + <span style="display:inline-flex;padding:0.05rem 0.35rem;border-radius:9999px;background:#FEE2E2;border:1px solid #FECACA;font-size:10px;color:#DC2626;">Vencido</span> → solo clientes con mora</span>
<span>→ Lista + esta semana → clientes con vencimientos próximos</span>
<span>→ Lista + <span style="display:inline-flex;padding:0.05rem 0.35rem;border-radius:9999px;background:#DBEAFE;border:1px solid #BFDBFE;font-size:10px;color:#1E40AF;">Pendiente</span> → clientes sin ningún pago</span>
</div>
</div>
</div>

  </tab>
</tabs>
