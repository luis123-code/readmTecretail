<span style="font-size:12px;font-weight:500;color:#249F65;text-transform:uppercase;letter-spacing:0.08em;">POS · Modo de Operación</span>

<span style="font-family:'Cormorant Garamond',Georgia,serif;font-style:italic;font-size:18px;color:#6B6B6B;line-height:1.6;display:block;margin-bottom:1.25rem;">Sistema de modos de operación para terminales POS que define la complejidad de interfaz y funcionalidades disponibles según el nivel de experiencia del operador y volumen de transacciones.</span>

<alert type="info">
  Dos modos disponibles: <strong>Normal (0)</strong> para operaciones completas y <strong>Fácil (1)</strong> para operaciones simplificadas. El cambio es inmediato y una tienda puede tener terminales en diferentes modos simultáneamente.
</alert>
