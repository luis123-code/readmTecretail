<span style="font-size:12px;font-weight:500;color:#249F65;text-transform:uppercase;letter-spacing:0.08em;">Analytics · Controles Temporales</span>

<span style="font-family:'Cormorant Garamond',Georgia,serif;font-style:italic;font-size:18px;color:#6B6B6B;line-height:1.6;display:block;margin-bottom:1.25rem;">Controles que permiten visualizar datos de Analytics (KPIs, gráficos, rankings) en períodos específicos. Incluye selectores de fecha para KPIs diarios, filtros de tipo y cantidad de períodos para gráficos, y vista mensual con selección de año/mes para métodos de pago.</span>

<alert type="info">
  Los filtros de fecha son <strong>fundamentales</strong> para el análisis temporal. Para KPIs, usa la <strong>vista diaria</strong> para monitoreo cotidiano y la <strong>vista mensual</strong> para análisis de períodos más largos. Para gráficos, usa <strong>períodos largos</strong> (años, meses) para ver <strong>tendencias claras</strong> y <strong>períodos cortos</strong> (días) para monitoreo de corto plazo. El tipo <strong>"Personalizado"</strong> es útil para analizar eventos específicos.
</alert>

## Cómo Cambiar Fecha en KPIs

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Proceso</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">1. Haz <strong>clic</strong> en el campo de fecha</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">2. Se abre un <strong>calendario</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">3. <strong>Navega</strong> al mes y año necesarios</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">4. <strong>Selecciona</strong> el día</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">5. Los KPIs se <strong>actualizan automáticamente</strong></p>
</div>
</div>

</div>
<div>

<div style="background:#D1FAE5;border:1px solid #A7F3D0;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#065F46;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Qué pasa al cambiar</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#047857;margin:0;line-height:1.5;">• KPIs se <strong>recalculan</strong> para la nueva fecha</p>
<p style="font-size:12px;color:#047857;margin:0;line-height:1.5;">• Muestra <strong>datos de esa fecha</strong> específica</p>
<p style="font-size:12px;color:#047857;margin:0;line-height:1.5;">• Datos se actualizan en <strong>tiempo real</strong></p>
</div>

## Filtros de Fecha para Gráficos

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Tipo de Período</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <strong>Dropdown</strong> con opciones</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Por defecto: <strong>Años</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Define <strong>cómo se agrupan</strong> los datos</p>
</div>
<div style="background:#ffffff;border:1px solid #E5E5E5;border-radius:0.5rem;padding:0.75rem;margin-top:0.5rem;">
<p style="font-size:11px;color:#6B6B6B;margin:0;"><strong>Opciones:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0.2rem 0 0;line-height:1.5;">• <strong>Años:</strong> Datos por año</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <strong>Meses:</strong> Datos por mes</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <strong>Días:</strong> Datos por día</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <strong>Personalizado:</strong> Rango específico</p>
</div>
</div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Cantidad de Períodos</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <strong>Campo numérico</strong> o dropdown</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Define <strong>cuántos períodos</strong> mostrar</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Por defecto: <strong>3 (años), 12 (meses), 30 (días)</strong></p>
</div>
<div style="background:#ffffff;border:1px solid #E5E5E5;border-radius:0.5rem;padding:0.75rem;margin-top:0.5rem;">
<p style="font-size:11px;color:#6B6B6B;margin:0;"><strong>Ejemplos:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0.2rem 0 0;line-height:1.5;">• Años: <strong>3, 5, 10</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Meses: <strong>6, 12, 24</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Días: <strong>7, 30, 90</strong></p>
</div>
</div>

</div>
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Fechas Personalizadas</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;"><strong>Solo para tipo "Personalizado":</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• <strong>"Desde":</strong> Fecha inicial</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• <strong>"Hasta":</strong> Fecha final</p>
</div>
<p style="font-size:12px;color:#6B6B6B;margin:0.5rem 0;line-height:1.5;"><strong>Cómo usar:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">1. Selecciona <strong>"Personalizado"</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">2. Clic en <strong>"Desde"</strong> + selecciona fecha inicial</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">3. Clic en <strong>"Hasta"</strong> + selecciona fecha final</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">4. Gráfico se <strong>actualiza automáticamente</strong></p>

## Diferencia entre Filtros

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Filtros de KPIs vs Filtros de Gráficos</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;"><strong>Filtros de KPIs:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• <strong>Fecha única</strong> (día específico)</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Vista <strong>instantánea</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Para <strong>monitoreo diario</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0.5rem 0 0;line-height:1.5;"><strong>Filtros de Gráficos:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• <strong>Rango de períodos</strong> (años, meses, días)</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Vista <strong>temporal</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Para <strong>análisis de tendencias</strong></p>
</div>
</div>

</div>
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Vista Diaria vs Vista Mensual</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;"><strong>Vista Diaria:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Un <strong>día específico</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Para <strong>monitoreo diario</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• KPIs estándar</p>
<p style="font-size:12px;color:#6B6B6B;margin:0.5rem 0 0;line-height:1.5;"><strong>Vista Mensual:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Un <strong>mes completo</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Para <strong>análisis mensual</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• KPIs de Métodos de Pago (año/mes)</p>
</div>
