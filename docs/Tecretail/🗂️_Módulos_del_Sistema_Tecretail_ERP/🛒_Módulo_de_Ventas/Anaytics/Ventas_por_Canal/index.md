<span style="font-size:12px;font-weight:500;color:#249F65;text-transform:uppercase;letter-spacing:0.08em;">Analytics · Análisis Multicanal</span>

<span style="font-family:'Cormorant Garamond',Georgia,serif;font-style:italic;font-size:18px;color:#6B6B6B;line-height:1.6;display:block;margin-bottom:1.25rem;">Visualización que muestra el desglose de ventas según el canal de venta utilizado (tienda física, online, mayorista, minorista, etc.). Permite identificar el canal principal, comparar rendimiento entre canales, calcular ticket promedio y optimizar la asignación de recursos.</span>

<alert type="info">
  Revisa esta visualización <strong>regularmente</strong> para identificar cambios en la distribución de ventas. <strong>Enfócate en los canales</strong> que generan más ventas y <strong>desarrolla</strong> los canales con potencial. Calcula el <strong>ticket promedio</strong> por canal (Ventas ÷ Pedidos) para entender la eficiencia de cada uno. Las ventas <strong>"Sin canal"</strong> indican datos incompletos que deben corregirse.
</alert>
