
<span style="font-size:12px;font-weight:500;color:#249F65;text-transform:uppercase;letter-spacing:0.08em;">Analytics · Personalización Visual</span>

<span style="font-family:'Cormorant Garamond',Georgia,serif;font-style:italic;font-size:18px;color:#6B6B6B;line-height:1.6;display:block;margin-bottom:1.25rem;">Checkboxes que permiten mostrar u ocultar diferentes líneas en el gráfico de ventas por período. Controla qué series visualizar (ventas, facturas, boletas, documentos combinados, cantidad de productos) para personalizar la visualización y realizar comparaciones específicas.</span>

<alert type="info">
  Activa <strong>solo las series que necesitas</strong> para evitar confusión visual. Usa la <strong>comparación de series</strong> para entender relaciones entre ventas, facturación y boletación. El <strong>switch de combinación</strong> de documentos es útil cuando no necesitas distinguir entre facturas y boletas. La serie <strong>"Cantidad de Productos"</strong> muestra cantidad (no monto), por lo que la escala del eje Y puede ser diferente. Compara montos con cantidades para entender el <strong>ticket promedio</strong>.
</alert>

## Cómo Usar el Control de Series

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 1: Activar una serie</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">1. Haz <strong>clic</strong> en el checkbox de la serie</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">2. El checkbox se <strong>marca</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">3. La <strong>serie aparece</strong> en el gráfico</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">4. La <strong>leyenda</strong> se actualiza</p>
</div>
<p style="font-size:11px;color:#86868b;margin:0.5rem 0 0;line-height:1.4;"><strong>Ejemplo:</strong> Clic en "Ventas" → línea azul aparece.</p>
</div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 2: Desactivar una serie</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">1. Haz <strong>clic</strong> en el checkbox marcado</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">2. El checkbox se <strong>desmarca</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">3. La <strong>serie desaparece</strong> del gráfico</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">4. La <strong>leyenda</strong> se actualiza</p>
</div>
<p style="font-size:11px;color:#86868b;margin:0.5rem 0 0;line-height:1.4;"><strong>Ejemplo:</strong> Clic en "Facturas" (marcado) → línea púrpura desaparece.</p>
</div>

</div>
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 3: Activar múltiples series</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Haz clic en <strong>varios checkboxes</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Las series marcadas <strong>aparecen</strong> en el gráfico</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Puedes <strong>comparar</strong> múltiples series a la vez</p>
</div>
<p style="font-size:11px;color:#86868b;margin:0.5rem 0 0;line-height:1.4;"><strong>Ejemplo:</strong> Activa "Ventas" + "Facturas" → ambas líneas visibles para comparar.</p>
</div>

</div>
</div>

## Cuándo Usar el Control de Series

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Para comparación</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Comparar <strong>ventas</strong> con <strong>facturación</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Comparar <strong>ventas</strong> con <strong>boletación</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Comparar <strong>montos</strong> con <strong>cantidades</strong></p>
</div>
<p style="font-size:11px;color:#86868b;margin:0.5rem 0 0;line-height:1.4;"><strong>Qué hacer:</strong> Activa las series que quieres comparar, observa diferencias, identifica patrones, toma decisiones.</p>
</div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Para simplificación</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Ver <strong>solo una serie</strong> para análisis enfocado</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Reducir <strong>ruido visual</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Evitar <strong>confusión</strong></p>
</div>
<p style="font-size:11px;color:#86868b;margin:0.5rem 0 0;line-height:1.4;"><strong>Qué hacer:</strong> Desmarca todas excepto una, el gráfico se simplifica, análisis más claro.</p>
</div>

</div>

## Diferencia con Otros Controles

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Control de Series vs Filtros de Estado</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;"><strong>Control de Series:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Controla <strong>qué series mostrar</strong> en el gráfico</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• <strong>No filtra datos</strong>, solo visualización</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Afecta <strong>solo el gráfico</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0.5rem 0 0;line-height:1.5;"><strong>Filtros de Estado:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Controla <strong>qué estados incluir</strong> en los datos</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• <strong>Filtra los datos</strong> antes de mostrar</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">• Afecta <strong>KPIs y gráficos</strong></p>
</div>
</div>

</div>
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Diferencias clave</p>
<div style="display:flex;flex-direction:column;gap:0.5rem;">
<div style="background:#FFFFFF;border:1px solid #E5E5E5;border-radius:0.5rem;padding:0.75rem;">
<p style="font-size:12px;font-weight:600;color:#121212;margin:0 0 0.3rem;">Visualización vs Datos</p>
<p style="font-size:11px;color:#6B6B6B;margin:0;line-height:1.4;"><strong>Control de Series</strong> solo cambia qué líneas ves. <strong>Filtros de Estado</strong> cambian qué datos se incluyen.</p>
</div>
<div style="background:#FFFFFF;border:1px solid #E5E5E5;border-radius:0.5rem;padding:0.75rem;">
<p style="font-size:12px;font-weight:600;color:#121212;margin:0 0 0.3rem;">Alcance</p>
<p style="font-size:11px;color:#6B6B6B;margin:0;line-height:1.4;"><strong>Control de Series</strong> solo afecta el gráfico. <strong>Filtros de Estado</strong> afectan toda la sección (KPIs + gráficos).</p>
</div>
</div>
</div>
