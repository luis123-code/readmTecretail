<alert type="info">
  Los filtros de estado son <strong>herramientas fundamentales</strong> para el análisis en Analytics. Por defecto <strong>excluyen los documentos anulados</strong> para no distorsionar los datos. Usa la <strong>selección múltiple</strong> para análisis más amplios. Los filtros se <strong>aplican automáticamente</strong>, no necesitas un botón de "Aplicar". Combina diferentes filtros para análisis más específicos.
</alert>

## Cómo Usar los Filtros

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 1: Abrir el dropdown</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Haz <strong>clic</strong> en el dropdown de estado que necesitas</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Se abre una <strong>lista con checkboxes</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Verás <strong>todos los estados</strong> disponibles</p>
</div>
</div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 2: Seleccionar los estados</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;"><strong>Para seleccionar:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">→ Clic en <strong>checkbox</strong> del estado</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">→ Checkbox se <strong>marca</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">→ Estado se <strong>agrega</strong> a la selección</p>
<p style="font-size:12px;color:#6B6B6B;margin:0.5rem 0 0;line-height:1.5;"><strong>Para deseleccionar:</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">→ Clic en <strong>checkbox marcado</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">→ Checkbox se <strong>desmarca</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;">→ Estado se <strong>elimina</strong></p>
</div>
</div>

</div>
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 3: Aplicar el filtro</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• El filtro se aplica <strong>automáticamente</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Los <strong>KPIs</strong> se actualizan en tiempo real</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Los <strong>gráficos</strong> se actualizan en tiempo real</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <strong>No necesitas</strong> botón de "Aplicar"</p>
</div>
</div>

</div>
</div>
