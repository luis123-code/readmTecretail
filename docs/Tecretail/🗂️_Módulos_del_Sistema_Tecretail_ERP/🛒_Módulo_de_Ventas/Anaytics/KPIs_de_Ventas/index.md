
<span style="font-size:12px;font-weight:500;color:#249F65;text-transform:uppercase;letter-spacing:0.08em;">Analytics · Indicadores de Rendimiento</span>

<span style="font-family:'Cormorant Garamond',Georgia,serif;font-style:italic;font-size:18px;color:#6B6B6B;line-height:1.6;display:block;margin-bottom:1.25rem;">Indicadores clave que muestran el rendimiento de ventas en tiempo real: monto total, cantidad de órdenes y promedio por venta. Incluye filtros por fecha, estado de ventas y tipo de nota para análisis detallado y toma de decisiones.</span>

<alert type="info">
  Revisa los KPIs de <strong>hoy cada mañana</strong> para tener una visión rápida del rendimiento. Usa el filtro de <strong>estado</strong> para ver solo ventas <span style="background:#22C55E;color:#fff;font-size:10px;padding:0.15rem 0.5rem;border-radius:4px;">Completadas</span> (ventas reales) y excluir <span style="background:#EF4444;color:#fff;font-size:10px;padding:0.15rem 0.5rem;border-radius:4px;">Anuladas</span>. Los KPIs incluyen <strong>todas las órdenes</strong>, independientemente de facturación.
</alert>

## Cómo Usar los KPIs

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 1: Seleccionar fecha</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Selector arriba de los KPIs</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Por defecto: <strong>fecha actual</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Clic en campo → abre <strong>calendario</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Navega a <strong>mes/año</strong> necesario</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Selecciona <strong>día</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• KPIs se <strong>recalculan automáticamente</strong></p>
</div>
</div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 2: Filtrar por Estado</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Dropdown <strong>"Estado de Ventas"</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <strong>Checkboxes</strong> para múltiples selección</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Por defecto: <strong>todos excepto "Anulada"</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• KPIs se <strong>actualizan automáticamente</strong></p>
</div>
</div>

</div>
<div>

<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.6rem;">Paso 3: Filtrar por Tipo de Nota</p>
<div style="display:flex;flex-direction:column;gap:0.3rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Dropdown <strong>"Tipo de Nota"</strong></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Opciones:</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;"><strong>• Todos:</strong> Todas las ventas</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;"><strong>• Con nota:</strong> Solo BM = 1</p>
<p style="font-size:12px;color:#6B6B6B;margin:0 0 0 1rem;line-height:1.5;"><strong>• Sin nota:</strong> BM = 0 o null</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• KPIs se <strong>actualizan automáticamente</strong></p>
</div>
</div>

</div>

<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Revisar <strong>ventas de hoy</strong></p>

<div style="display:flex;flex-direction:column;gap:0.3rem;">
