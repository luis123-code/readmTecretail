## Fusionar Inventarios

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>
<p style="font-size:13px;color:#6B6B6B;line-height:1.7;margin-bottom:1rem;">Combina registros duplicados del mismo producto dentro de un mismo almacén en un único inventario consolidado.</p>
<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.75rem;">Funcionalidades</p>
<div style="display:flex;flex-direction:column;gap:0.4rem;">
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Busca productos por <span style="font-weight:500;color:#121212;">nombre</span>, <span style="font-weight:500;color:#121212;">código de barras</span> o <span style="font-weight:500;color:#121212;">SKU</span></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• Muestra registros <span style="font-weight:500;color:#121212;">agrupados por almacén</span></p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <span style="font-weight:500;color:#121212;">Fusiona</span> los stocks seleccionados en un solo registro</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <span style="font-weight:500;color:#121212;">Elimina</span> registros duplicados automáticamente</p>
<p style="font-size:12px;color:#6B6B6B;margin:0;line-height:1.5;">• <span style="font-weight:500;color:#121212;">Reasigna</span> órdenes vinculadas al nuevo registro consolidado</p>

## Fusión Masiva

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1.5rem;align-items:start;margin:1.25rem 0;">
<div>
<p style="font-size:13px;color:#6B6B6B;line-height:1.7;margin-bottom:1rem;">Fusiona múltiples productos simultáneamente mediante un archivo Excel.</p>
<div style="background:#F5F5F5;border:1px solid #E5E5E5;border-radius:0.75rem;padding:1rem;margin-bottom:0.75rem;">
<p style="font-size:11px;font-weight:600;color:#121212;text-transform:uppercase;letter-spacing:0.05em;margin:0 0 0.75rem;">Pasos</p>
<div style="display:flex;flex-direction:column;gap:0.4rem;">
<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#E5E5E5;color:#6B6B6B;flex-shrink:0;">01</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">Selecciona el <span style="font-weight:500;color:#121212;">almacén</span> donde se realizará la fusión</span></div>
<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#E5E5E5;color:#6B6B6B;flex-shrink:0;">02</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">Sube un archivo Excel con: <span style="font-family:'JetBrains Mono',monospace;background:#F5F5F5;padding:0 0.25rem;border-radius:3px;border:1px solid #E5E5E5;">código_barras</span> y <span style="font-family:'JetBrains Mono',monospace;background:#F5F5F5;padding:0 0.25rem;border-radius:3px;border:1px solid #E5E5E5;">cantidad_final</span></span></div>
<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#E5E5E5;color:#6B6B6B;flex-shrink:0;">03</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">El sistema muestra una <span style="font-weight:500;color:#121212;">vista previa</span> antes de confirmar</span></div>
<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#A7F3D0;color:#065F46;flex-shrink:0;">04</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">Fusiona automáticamente registros duplicados</span></div>
<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#A7F3D0;color:#065F46;flex-shrink:0;">05</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">Ajusta el <span style="font-weight:500;color:#121212;">stock final</span> según el Excel</span></div>
<div style="display:flex;gap:0.75rem;align-items:flex-start;"><span style="font-family:'JetBrains Mono',monospace;font-size:10px;padding:0.1rem 0.35rem;border-radius:4px;background:#A7F3D0;color:#065F46;flex-shrink:0;">06</span><span style="font-size:12px;color:#6B6B6B;line-height:1.5;">Reasigna <span style="font-weight:500;color:#121212;">órdenes vinculadas</span> al nuevo registro</span></div>
